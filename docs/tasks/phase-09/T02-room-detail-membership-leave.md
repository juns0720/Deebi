# P09-T02 — 룸 상세, 멤버십, 나가기 API

## 목표

룸 상세 조회와 멤버십 검증, 룸 나가기 흐름을 구현한다.

## 변경 범위

- **해도 됨:** `GET /api/rooms/[code]`, `POST /api/rooms/[code]/leave`, membership helper
- **하면 안 됨:** 채팅 API, 룸 삭제, 복잡한 권한 관리

## 작업

- room code로 룸 상세를 조회한다.
- 룸 멤버만 상세 정보를 볼 수 있게 한다.
- 멤버 목록을 health 순으로 정렬한다.
- 룸 나가기 API를 구현한다.
- 나간 룸은 내 목록에서 빠지게 한다.

## 완료 조건

- [ ] 룸 멤버만 상세 조회가 가능하다.
- [ ] 멤버가 health 순으로 정렬된다.
- [ ] 룸 나가기가 동작한다.
- [ ] 룸 나가기 후 목록에서 제거된다.
- [ ] 룸 삭제 기능은 없다.

## 개발자 테스트

1. 룸 멤버로 상세를 조회한다.
2. 룸 멤버가 아닌 사용자로 상세 조회가 거부되는지 확인한다.
3. 룸 나가기 후 목록에서 사라지는지 확인한다.
