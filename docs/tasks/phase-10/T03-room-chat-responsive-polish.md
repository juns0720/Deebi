# P10-T03 — 룸과 채팅 반응형 polish

## 목표

공동 룸과 채팅이 모바일/데스크톱에서 캐릭터와 대화를 모두 자연스럽게 보여주도록 다듬는다.

## 변경 범위

- **해도 됨:** room layout, member placement, chat panel/sheet, overflow 처리
- **하면 안 됨:** 채팅 고급 기능 추가, room API 계약 변경

## 작업

- 1~8명 멤버 배치가 안정적인지 확인한다.
- 채팅 패널이 방 스테이지를 가리지 않게 한다.
- 모바일 하단 시트 높이와 닫기 UX를 점검한다.
- 긴 메시지와 긴 login overflow를 수정한다.

## 완료 조건

- [ ] 8명 룸에서도 캐릭터와 라벨이 식별된다.
- [ ] 데스크톱 채팅 패널이 스테이지를 가리지 않는다.
- [ ] 모바일 채팅 시트가 열리고 닫힌다.
- [ ] 긴 메시지가 레이아웃을 깨지 않는다.

## 개발자 테스트

1. 8명 mock 또는 실제 룸을 데스크톱에서 확인한다.
2. 모바일 폭에서 채팅 시트를 확인한다.
3. 긴 login과 긴 메시지를 테스트한다.
