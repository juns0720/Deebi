# P08-T03 — 장착/해제 API

## 목표

사용자가 보유한 아이템만 슬롯에 장착하거나 해제할 수 있는 API를 만든다.

## 변경 범위

- **해도 됨:** `/api/inventory/equip`, 장착/해제 write, 권한 검증
- **하면 안 됨:** gacha reveal UI, room 기능

## 작업

- 세션 사용자 기준으로 장착 API를 만든다.
- 보유하지 않은 아이템 장착을 거부한다.
- 슬롯당 1개만 장착되게 한다.
- null 또는 action으로 해제를 처리한다.

## 완료 조건

- [ ] 보유 아이템만 장착 가능하다.
- [ ] 슬롯당 1개만 장착된다.
- [ ] 해제가 가능하다.
- [ ] 다른 사용자의 inventory를 조작할 수 없다.

## 개발자 테스트

1. 보유 아이템을 장착한다.
2. 같은 슬롯의 다른 아이템을 장착해 교체되는지 확인한다.
3. 미보유 아이템 장착이 거부되는지 확인한다.
