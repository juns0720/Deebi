# P08-T04 — 뽑기와 장착 UI 연결

## 목표

사용자가 dashboard에서 뽑기 결과를 보고 아이템을 장착/해제할 수 있게 한다.

## 변경 범위

- **해도 됨:** gacha button, result reveal, inventory interaction, equipped UI
- **하면 안 됨:** 결제, 상점, 아이템 거래

## 작업

- 포인트 부족/가능 상태를 UI에 표시한다.
- 뽑기 버튼과 결과 표시를 연결한다.
- 새 아이템/중복 환급 메시지를 구분한다.
- inventory에서 장착/해제를 실행한다.
- 캐릭터 렌더러에 결과가 반영되게 한다.

## 완료 조건

- [ ] 100포인트 미만이면 버튼 상태가 명확하다.
- [ ] 뽑기 결과가 읽힌다.
- [ ] 중복 환급 30포인트가 UI에 보인다.
- [ ] 장착/해제 후 캐릭터 표시가 바뀐다.
- [ ] loading/error 상태가 있다.

## 개발자 테스트

1. 포인트 부족 상태에서 뽑기 버튼을 확인한다.
2. 뽑기를 실행하고 결과를 확인한다.
3. 결과 아이템을 장착해 캐릭터 변화를 확인한다.
