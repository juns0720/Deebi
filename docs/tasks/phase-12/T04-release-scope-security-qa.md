# P12-T04 — MVP 범위와 보안 최종 QA

## 목표

MVP 제외 기능이 들어가지 않았고, 출시 전 보안/광고/결제 경계가 지켜졌는지 최종 확인한다.

## 변경 범위

- **해도 됨:** scope cleanup, 보안 QA 수정, 문서 정리
- **하면 안 됨:** 결제, 실제 광고 SDK, 상점, 친구 요청, DM 추가

## 작업

- MVP 제외 항목 구현 여부를 검색한다.
- 실제 광고 SDK나 결제 코드가 없는지 확인한다.
- secret/token 노출을 다시 점검한다.
- 핵심 사용자 흐름을 확인한다.

## 완료 조건

- [ ] MVP 제외 기능이 구현되어 있지 않다.
- [ ] 실제 광고 SDK가 없다.
- [ ] 결제/상점/거래 코드가 없다.
- [ ] secret/token 노출이 없다.
- [ ] 최종 QA 결과가 기록된다.

## 개발자 테스트

1. MVP 제외 항목 목록을 기준으로 화면과 코드를 확인한다.
2. 브라우저에서 로그인, 대시보드, 뽑기, 룸 흐름을 확인한다.
