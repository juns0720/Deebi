# P07-T04 — 방치 모드, 문, 룸 상태 신호

## 목표

내 방에서 방치 모드와 공동 룸 진입 오브젝트가 실제 데이터 기반으로 자연스럽게 보이게 한다.

## 변경 범위

- **해도 됨:** idle mode UI, door CTA, joined room mini signal, 광고 예약 영역
- **하면 안 됨:** 실제 광고 SDK, room 생성/참여 API 구현

## 작업

- 방치 모드 진입/종료 UI를 연결한다.
- 종료 버튼이 항상 보이게 한다.
- 문 오브젝트를 함께하는 방 탭 또는 최근 룸 진입 CTA로 연결한다.
- 참여 중인 룸 상태를 작은 방 안 오브젝트로 표시한다.
- 광고 예약 영역은 과하게 드러나지 않게 둔다.

## 완료 조건

- [ ] 방치 모드에서 방 스테이지가 커진다.
- [ ] 종료 버튼이 항상 보인다.
- [ ] 문 오브젝트가 공동 룸 진입 맥락으로 보인다.
- [ ] 룸 상태 신호가 방 안 오브젝트처럼 보인다.
- [ ] 실제 광고 SDK가 없다.

## 개발자 테스트

1. 내 방에서 방치 모드를 켜고 끈다.
2. 참여 중인 룸이 있을 때 미니 신호가 보이는지 확인한다.
3. 광고 영역이 사용자 경험을 방해하지 않는지 확인한다.
