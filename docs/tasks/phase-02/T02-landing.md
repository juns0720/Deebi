# P02-T02 — 랜딩 정적 프로토타입

## 목표

로그인 전 첫 화면에서 Deebi의 핵심 문장과 공동 룸 감성을 즉시 보여주는 픽셀 랜딩을 만든다.

## 변경 범위

- **해도 됨:** `src/app/page.tsx`, landing components, mock preview data, CSS
- **하면 안 됨:** 실제 GitHub OAuth 연결, 대시보드 구현, 실제 회원 상태 분기

## 작업

- H1을 `각자 코딩해도, 같은 방에 있는 것처럼.`으로 구성한다.
- 보조 문장과 GitHub 시작 CTA를 배치한다.
- 공동 작업 테이블 룸 미리보기를 픽셀 그래픽 감성으로 만든다.
- 내 방, 뽑기, 함께하는 방의 3단계 루프 힌트를 보여준다.
- OAuth 미구현 상태에서도 버튼은 안전한 준비 상태 또는 mock CTA로 동작한다.

## 완료 조건

- [ ] 첫 viewport에서 Deebi 이름과 핵심 문장이 보인다.
- [ ] 공동 룸 미리보기가 실제 서비스 경험을 암시한다.
- [ ] 전체 랜딩이 PF스타더스트와 픽셀 UI 토큰을 사용한다.
- [ ] MVP 제외 기능을 가능한 것처럼 홍보하지 않는다.
- [ ] 모바일 360px에서 텍스트가 겹치지 않는다.
- [ ] `npm run lint`, `npm run typecheck`, `npm run build`가 통과한다.

## 개발자 테스트

1. `/`를 데스크톱 폭에서 확인한다.
2. `/`를 모바일 360px 폭에서 확인한다.
3. 첫 화면만 보고 “친구들과 같은 방에서 개발하는 서비스”인지 느껴지는지 판단한다.
